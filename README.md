# 🎓 Student Performance Analysis

## Overview

This project analyzes factors affecting student academic performance using Python.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dashboard

![Dashboard](dashboard.png)

## Key Insights

- Female students performed slightly better than male students.
- Test preparation significantly improved scores.
- Students studying more than 10 hours achieved the highest average scores.
- Higher parental education was associated with better performance.
- Reading and writing scores showed a strong positive correlation.
